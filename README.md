# Java 学习周1 项目
这是我第一周的 Java 入门学习仓库，记录了从基础语法到综合项目的所有练习代码。

##  项目结构
src/└── week1/
├── day1/ # 入门与输出
├── day2/ # 变量、数据类型与基础输入输出
├── day3/ # 分支与条件判断
├── day4/ # 循环结构
├── day5/ # 数组基础
├── day6/ # 数组操作
└── day7/ # 综合项目：学生成绩录入系统

## 主要功能
- 基础输入输出、变量与数据类型练习
- 分支结构（if-else、switch）与循环结构（for、while）练习
- 数组的定义、遍历、求和、排序等操作
- **day7 综合项目：学生成绩录入系统**
  - 录入学生姓名与成绩
  - 计算平均分、最高分、最低分
  - 统计成绩分布（优秀/良好/及格/不及格）

##  如何运行
1. 下载或克隆仓库到本地
2. 用 IntelliJ IDEA 打开项目
3. 找到 `src/week1/dayX` 下的 `.java` 文件，右键运行即可

##  学习总结
第一周重点掌握了 Java 基础语法、流程控制与数组，为后续面向对象编程打下了基础。
