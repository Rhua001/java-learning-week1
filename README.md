##必学知识点
##类与对象
类：对象的模板，封装了属性和行为
对象：类的实例，通过 new 关键字创建
构造方法：无参 / 有参，用于初始化对象
this：指向当前对象的引用，区分成员变量和局部变量
## 封装
使用 private 隐藏成员变量
通过 public 的 getter/setter 方法提供访问
好处：提高安全性、可维护性，隐藏内部实现细节
## 继承
extends 关键字实现类与类之间的继承
子类拥有父类的所有非私有成员（构造器除外）
super：调用父类构造器或方法
方法重写（Override）：子类重新定义父类方法，运行时多态的基础
## 多态
前提：继承 + 方法重写 + 父类引用指向子类对象
向上转型：Animal a = new Dog();（自动）
向下转型：Dog d = (Dog) a;（需要 instanceof 判断）
动态绑定：调用方法时根据实际对象类型决定执行哪个方法
## 抽象类
abstract class 定义
可包含抽象方法（没有方法体）和普通方法、成员变量、构造器
不能实例化，必须被子类继承并实现所有抽象方法
## 接口
interface 定义
方法默认 public abstract（JDK 8 后可以有 default 和 static 方法）
一个类可以 implements 多个接口
接口是“契约”，抽象类是“模板”
